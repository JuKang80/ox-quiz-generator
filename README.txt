OX 퀴즈 자동 생성기

학습 노트에서 문장을 자동 추출해 O/X 퀴즈를 생성하고, docx 형식으로 저장하는 Python 기반 툴입니다.
반복적인 요약 학습보다 기억 인출 기반 학습(Retrieval-based Learning)이 더 효과적이라는 심리학적 근거에 기반하여 개발되었습니다.

---

[주요 기능]

- txt, md 형식의 노트 파일 불러오기
- 문장 단위 자동 추출 및 퀴즈 변환
- 모든 문장을 기본적으로 '정답: O(True)'로 처리
- 퀴즈 출력 결과를 docx 파일로 저장
- 입력 오류, 빈 파일 등 예외 상황 처리

---

[실행 방법]

1. Python 3.x 환경 필요
2. 아래 명령어로 필요한 라이브러리 설치

   pip install python-docx

3. main.py 실행

   python main.py

4. 콘솔 안내에 따라 텍스트 파일을 입력하면,
   퀴즈가 자동 생성되어 docx 파일로 저장됩니다.
